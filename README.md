# Portfolio Setup and Customization Instructions

## Prerequisites
- [Node.js](https://nodejs.org/) (version 14 or higher)
- [npm](https://www.npmjs.com/) (Node package manager)

## Steps to Set Up Your Portfolio

1. **Clone the repository**:
   ```bash
   git clone https://github.com/deepaghimire/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Customize your portfolio**:
   - Open the project in your favorite code editor.
   - Navigate to the `src` directory.
   - Edit `config.js` to update details like your name, bio, and social media links.
   - Add your projects and relevant sections in `projects.js`.

4. **Run the development server**:
   ```bash
   npm start
   ```
   Your portfolio should now be running on `http://localhost:3000`.

## Deployment
- To deploy your portfolio, build the project using:
   ```bash
   npm run build
   ```
- Follow the deployment instructions for the hosting service of your choice, like Netlify or GitHub Pages.
